# Area finding program

## How to use?
The easy way is to use is run main.exe
I complied only windows 
If you on mac or linux, you can complie it by yourself
```bash
clang main.c -o main
```
or 
```bash
gcc main.c -o main
```
<br/>

## To run it
<br/>
in terminal

```bash
./main
``` 

or in powershell

```bash
./main.exe
```
